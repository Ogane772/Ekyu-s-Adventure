1：下記「Microsoft Visual Studio ダウンロードページ」へアクセスしてください。

https://visualstudio.microsoft.com/ja/downloads/

2：ページ内下部にあります「その他のツールとフレームワーク」をクリックしてください。

3：プルダウンメニュー内の「Visual Studio 2019 の Microsoft Visual C++ 再頒布可能パッケージ」の項目にある「x64」にチェックを入れてダウンロードを選択してください。

4：ダウンロードされた「VC_redist.x86.exe」をダブルクリックし、案内に従ってインストールしてください。
   インストール完了後、ゲームの起動をお試しください
